Requirements to tilemap

ONLY ONE SPACE INDENTS
NO EMPTY STRINGS
EDGES w** | a**
SIZE N*N

Designations
* = {0..9}
? = Whatever

v?? Void
w** Wall -- ** - texture number
a** Animated Wall -- ** - texture number
p?? Player
e** Enemy -- ** - enemy type
d** Destr. wall -- ** - texture number
i** Item -- ** - item type
b** Background -- ** - texture number, b00 - level end
o** Object -- ** - texture number
