First-person Shooter.

Using [SFML](https://www.sfml-dev.org/download/sfml/2.4.2/)

Using Glasgow Haskell Compiler

Using GNU MAKE

Using GNU Compiler Collection

## Build

To build use:

```
make force
```

## Execute

To execute use:

```
make run
```

## Clean

To clean use:

```
make clean
```
